# LUMEJ (Ethical and Fair Use License for Free Code)

A source-available license focused on weak copyleft, strict user privacy, and the explicit prohibition of abusive practices such as invasive adware and unnecessary telemetry.

## How to Adopt

To apply the LUMEJ license to your project:

1. Copy the full text from the `LICENSE.txt` (English) or `LICENSE_PT.txt` (Portuguese) file.
2. Create a file named `LICENSE` (or `LICENSE.txt`) at the root of your repository.
3. Replace the `[Year]` and `[Copyright Holder's Name]` placeholders with your own information.

## Core Pillars

* **Commercial Freedom:** You are allowed to sell, modify, and distribute the software or derivative works, provided all license terms are respected.
* **Weak Copyleft:** Core modifications, bug fixes, or direct optimizations to the original files must remain open-source under this same license. Separate plugins, modules, or extensions interacting via APIs/dynamic linking are exempt.
* **Anti-Adware & Anti-Telemetry:** Strictly prohibits hidden or excessive data collection. Intrusive full-screen ads, aggressive pop-ups, or any advertising that disrupts the core application flow are forbidden.
